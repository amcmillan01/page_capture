1. Install the extension from the Google Webstore:
      https://chrome.google.com/webstore/detail/jalljeamdfcpcigocpbgfbebdjfmpdof
  2. Inside your application
    • Create a queue to push your api commands to
      Q: Why do i need to do this?
      A: This is the queue that the "Page Capture" extension looks for, and it safe guards your application if the extension is not installed
        <script>
    // this MUST be on the top window scope
    var pcQueue = pcQueue || [];
  </script>
    • Now, let's get the version number of the extension
        <button id="get_version">run</button>

        <script>
    pcQueue.push(function() {
      document.getElementById('get_version').onclick = function() {
        PageCapture.getVersion(function(v) {
          console.log('version = ' + v);
        });
      };
    });
  </script>
    • Take a screenshot of an element on the page, now we're talking...
        <script>
    pcQueue.push(function() {
      PageCapture.captureElement('#some_element_id', function(base64Img) {
        var img = new Image();
        img.src = base64Img;
        document.body.appendChild(img);
      });
    });
  </script>
