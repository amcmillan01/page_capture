• getVersion(cb): get the version number
    • cb: callback function
      • version: the version number
  • capturePage(cb): capture the entire page of a given url
    • cb: callback function
      • img: the base64 image
  • captureElement(elementId, cb): capture an element on a given url
    • elementId: the element selector
    • cb: callback function
      • img: the base64 image
  • captureSection(x, y, width, height, cb): capture a section a given url
    • x: the starting x position
    • y: the starting y position
    • width: the width of the selection
    • height: the hieight of the selection
    • cb: callback function
      • img: the base64 image
  • captureImage(imgUrl, width, height, cb): capture an external image
    • imgUrl: the image url
    • width: the width of the image (if 0, width will be auto)
    • height: the height of the image (if 0, height will be auto)
    • cb: callback function
      • img: the base64 image
  • captureUrl(url, cb): capture an external page
    • url: the url of the page to capture
    • cb: callback function
      • img: the base64 image
  • captureUrlWithOverlay(url, imgUrl, includeOverlay, cb): capture an external page with an overlay
    • url: the url of the page to capture
    • imgUrl: the image the will be overlayed on "url"
    • includeOverlay: default is true
      if true, "imgUrl" will be part of the screenshot
      if false, "imgUrl" will NOT be part of the screenshot, the overlay config will be return
    • cb: callback function
      • data: the result with overlay data
        • img: the base64 image
        • zoomFactor: the zoom level
        • info: the overlay info 
          {
  "img": "data:image/png;base64,iVBOR...=",
  "zoomFactor": 1,
  "info": {
    "url": "http://www.yieldbot.com/",
    "overlay": {
      "top": "199",
      "left": "373",
      "size": "450x324",
      "width": "450",
      "height": "324",
      "url": "http://some.image.com/page.png"
    }
  }
}
